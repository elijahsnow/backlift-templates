To get started creating some blog entries, just add a file to your posts folder. You can use normal html files or markdown files that end with a ".md" extension.

In order to adjust the design of the website, look in the layouts folder. For example, changing the base layout will make all pages on the site look different.

The meta block

At the top of each post you'll want to add some metadata like the post title, created date and author. This is used later to sort the pages and generate a list of links to the posts.

Here's what the top of each new post should look like:

{{$ raw }}{{$ meta }}
title: Getting started
created: 2014-09-24
author: John Doe
public: yes
{{$ endmeta }}

{{$ layout /layouts/blogentry.html as content }}{{$ endraw }}

The content

The content of each page can be regular html or markdown.

Check out the markdown examples post to see a few quick examples of what you can do with markdown.
